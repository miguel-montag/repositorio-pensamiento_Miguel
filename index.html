Introducción

Contexto general del portafolio.

Este portafolio digital tiene como propósito reunir las evidencias principales de la materia Pensamiento de Ingeniería. A través de estas actividades se trabajaron temas relacionados con programación, inteligencia artificial, diseño, fabricación digital y mecanismos.

La página fue creada con HTML y CSS para organizar de forma clara cada tarea, mostrando el objetivo, el proceso seguido, las herramientas utilizadas y los aprendizajes obtenidos.

Objetivo del portafolio

Propósito principal del sitio web.

El objetivo de este portafolio es demostrar el avance y aprendizaje obtenido durante el curso, integrando evidencias de cada actividad y mostrando cómo los conceptos de ingeniería pueden aplicarse en proyectos prácticos.

Aprendizajes obtenidos

Habilidades desarrolladas durante las actividades.

  • Aprendí a organizar información dentro de una página web usando HTML y CSS.
  • Comprendí cómo la inteligencia artificial puede apoyar en la creación de proyectos.
  • Conocí procesos básicos de fabricación digital como impresión 3D, corte láser y CNC.
  • Desarrollé una mejor comprensión sobre la importancia del diseño, la planeación y la precisión.
  • Mejoré mi capacidad para documentar procesos y presentar evidencias de forma ordenada.

Conclusión

Cierre general del proyecto.

En conclusión, este portafolio representa el trabajo realizado durante la materia de Pensamiento de Ingeniería. Cada tarea permitió aplicar conocimientos de manera práctica y entender mejor cómo se conectan la tecnología, el diseño y la ingeniería.

Además, la creación de esta página web ayudó a fortalecer habilidades digitales, de organización y documentación, las cuales son importantes para futuros proyectos académicos y profesionales.
