Tarea 1 — Videojuego

En esta actividad se desarrolló un videojuego utilizando herramientas digitales y apoyo de inteligencia artificial. El objetivo fue comprender cómo se pueden integrar elementos de programación, diseño y lógica para crear una experiencia interactiva.

Entrega: Semana 1
Equipo: Individual
Herramientas: HTML, IA y GitHub Pages

Descripción y desarrollo

Durante esta actividad se desarrolló un videojuego básico utilizando herramientas digitales y apoyo de inteligencia artificial. Primero se planteó la idea principal del juego y posteriormente se trabajó en la lógica, el diseño visual y la estructura general.

Además, se utilizó GitHub Pages para publicar el proyecto y poder acceder al videojuego desde cualquier navegador web. Esta actividad permitió comprender mejor cómo se relacionan la programación, el diseño y la creatividad dentro de la ingeniería.
